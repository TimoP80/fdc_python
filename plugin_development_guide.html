1. Introduction

Welcome to the Fallout Dialogue Creator 2.0 Plugin Development Guide. This comprehensive documentation provides everything you need to create powerful plugins that extend the functionality of the Fallout Dialogue Creator application.

What is the Plugin System?

The plugin system is a modular architecture that allows developers to extend the Fallout Dialogue Creator without modifying the core application code. Plugins can:

  • Add new export formats for dialogue data
  • Integrate with external tools and services
  • Create custom UI elements and menus
  • Process dialogue in specialized ways
  • Add new scripting capabilities
  • Implement custom parsers for dialogue file formats

Plugin Types

TypeDescriptionUse Case
UI_EXTENSIONAdds UI elements, menus, dialogsCustom tools, visualizations
DIALOGUE_PROCESSORProcesses dialogue dataValidation, transformation
PARSER_EXTENSIONAdds file format supportImport/export formats
SCRIPTING_EXTENSIONExtends scripting capabilitiesCustom commands, functions
TESTING_EXTENSIONAdds testing functionalityDialogue testing, validation
EXPORT_EXTENSIONExports dialogue to formatsSSL, MSG, custom formats
Note: The plugin system is designed for Python 3.8+ and uses PyQt6 for UI components. All plugins must follow the PluginInterface contract.

PluginManager

The PluginManager class is the central hub for all plugin operations. It handles discovery, loading, lifecycle management, and hook dispatching.

PluginInterface

The PluginInterface is the base class that all plugins must inherit from:

class PluginInterface:
    def __init__(self):
        self.plugin_info = PluginInfo(...)

    def initialize(self, plugin_manager) -> bool:
        """Called when plugin is first loaded"""
        return True

    def activate(self) -> bool:
        """Called when plugin becomes active"""
        return True

    def deactivate(self) -> bool:
        """Called when plugin is deactivated"""
        return True

    def shutdown(self):
        """Called when plugin is unloaded"""
        pass

    def get_hooks(self) -> Dict[str, List[Callable]]:
        """Return hook functions for events"""
        return {}

Plugin Lifecycle

# Plugin Lifecycle States

Tutorial 1: Basic Plugin

Create a simple plugin that responds to events and adds a menu item.

"""
My First Plugin for Fallout Dialogue Creator
"""

from core.plugin_system import PluginInterface, PluginType, PluginHooks, PluginInfo


class MyFirstPlugin(PluginInterface):
    """A simple example plugin"""

    def __init__(self):
        super().__init__()
        
        self.plugin_info = PluginInfo(
            name="My First Plugin",
            version="1.0.0",
            description="A simple demonstration plugin",
            author="Your Name",
            plugin_type=PluginType.UI_EXTENSION
        )
        
        self.menu_action = None

    def initialize(self, plugin_manager):
        print(f"Initializing {self.plugin_info.name} v{self.plugin_info.version}")
        return True

    def activate(self):
        print(f"Activating {self.plugin_info.name}")
        return True

    def deactivate(self):
        print(f"Deactivating {self.plugin_info.name}")
        if self.menu_action:
            self.menu_action.deleteLater()
            self.menu_action = None
        return True

    def shutdown(self):
        print(f"Shutting down {self.plugin_info.name}")

    def get_hooks(self):
        return {
            PluginHooks.APP_STARTUP: [self.on_app_startup],
            PluginHooks.UI_MENU_BAR_CREATED: [self.on_menu_bar_created],
            PluginHooks.DIALOGUE_LOADED: [self.on_dialogue_loaded],
        }

    def on_app_startup(self, app):
        print("MyFirstPlugin: Application has started!")

    def on_menu_bar_created(self, menu_bar):
        print("MyFirstPlugin: Adding menu item")
        tools_menu = None
        for action in menu_bar.actions():
            if action.text() == "&Tools":
                tools_menu = action.menu()
                break

        if tools_menu:
            tools_menu.addSeparator()
            self.menu_action = tools_menu.addAction("My Plugin")
            self.menu_action.triggered.connect(self.show_welcome)

    def on_dialogue_loaded(self, dialogue):
        print(f"MyFirstPlugin: Loaded dialogue for {dialogue.npcname}")
        print(f"  Nodes: {dialogue.nodecount}")

    def show_welcome(self):
        from PyQt6.QtWidgets import QMessageBox
        QMessageBox.information(
            None,
            "My First Plugin",
            "Hello from my first plugin!\n\nThis demonstrates the basic plugin structure."
        )
Important: Plugin filenames must be lowercase with underscores for proper discovery.

Tutorial 2: Export Plugin

Create a plugin that exports dialogue data to JSON format.

"""
JSON Export Plugin for Fallout Dialogue Creator
Exports dialogue data to JSON format.
"""

import json
from pathlib import Path
from datetime import datetime

from core.plugin_system import PluginInterface, PluginType, PluginHooks, PluginInfo


class JSONExportPlugin(PluginInterface):
    """Plugin for exporting dialogues to JSON format"""

    def __init__(self):
        super().__init__()
        self.plugin_info = PluginInfo(
            name="JSON Export Plugin",
            version="1.0.0",
            description="Export dialogue data to JSON format",
            author="Your Name",
            plugin_type=PluginType.EXPORT_EXTENSION
        )
        self.menu_action = None
        self.current_dialogue = None

    def initialize(self, plugin_manager):
        return True

    def activate(self):
        return True

    def deactivate(self):
        if self.menu_action:
            self.menu_action.deleteLater()
            self.menu_action = None
        return True

    def get_hooks(self):
        return {
            PluginHooks.UI_MENU_BAR_CREATED: [self.on_menu_bar_created],
            PluginHooks.DIALOGUE_LOADED: [self.on_dialogue_loaded],
            PluginHooks.DIALOGUE_SAVED: [self.on_dialogue_saved],
        }

    def on_menu_bar_created(self, menu_bar):
        for action in menu_bar.actions():
            if action.text() == "&Tools":
                tools_menu = action.menu()
                tools_menu.addSeparator()
                export_menu = tools_menu.addMenu("Export to JSON")
                self.menu_action = export_menu.addAction("Export Current Dialogue")
                self.menu_action.triggered.connect(self.export_dialogue)
                break

    def on_dialogue_loaded(self, dialogue):
        self.current_dialogue = dialogue

    def on_dialogue_saved(self, dialogue, file_path):
        self.current_dialogue = dialogue

    def export_dialogue(self):
        if not self.current_dialogue:
            from PyQt6.QtWidgets import QMessageBox
            QMessageBox.warning(None, "Export Error", "No dialogue loaded")
            return

        try:
            dialogue_data = self._dialogue_to_dict(self.current_dialogue)

            from PyQt6.QtWidgets import QFileDialog
            default_name = f"{self.current_dialogue.npcname}.json"
            file_path, _ = QFileDialog.getSaveFileName(
                None,
                "Export Dialogue to JSON",
                default_name,
                "JSON Files (*.json);;All Files (*.*)"
            )

            if file_path:
                with open(file_path, 'w', encoding='utf-8') as f:
                    json.dump(dialogue_data, f, indent=2, ensure_ascii=False)

                from PyQt6.QtWidgets import QMessageBox
                QMessageBox.information(None, "Export Complete", f"Dialogue exported to:\n{file_path}")

        except Exception as e:
            from PyQt6.QtWidgets import QMessageBox
            QMessageBox.critical(None, "Export Error", f"Failed to export:\n{str(e)}")

    def _dialogue_to_dict(self, dialogue):
        result = {
            "metadata": {
                "npc_name": dialogue.npcname,
                "location": dialogue.location,
                "node_count": dialogue.nodecount,
                "export_date": datetime.now().isoformat()
            },
            "nodes": [],
            "variables": [],
            "custom_procedures": []
        }

        for node in dialogue.nodes:
            node_dict = {
                "name": node.nodename,
                "text": node.npctext,
                "text_female": node.npctext_female,
                "is_starting_node": node.is_wtg,
                "options": []
            }

            for option in node.options:
                option_dict = {
                    "text": option.optiontext,
                    "target_node": option.nodelink,
                    "has_skill_check": option.has_skill_check
                }
                
                if option.has_skill_check and option.skill_check:
                    option_dict["skill_check"] = {
                        "skill": option.skill_check.check_what.name,
                        "required_value": option.skill_check.required_value
                    }
                
                node_dict["options"].append(option_dict)
            
            result["nodes"].append(node_dict)

        for var in dialogue.variables:
            result["variables"].append({
                "name": var.name,
                "value": var.value
            })

        return result

Tutorial 3: Dialogue Validator Plugin

Create a plugin that validates dialogues for common issues.

"""
Dialogue Validator Plugin
Validates dialogues for common issues.
"""

from collections import Counter
from core.plugin_system import PluginInterface, PluginType, PluginHooks, PluginInfo


class ValidationResult:
    def __init__(self):
        self.errors = []
        self.warnings = []
    
    def add_error(self, message, node_name=None):
        location = f" [{node_name}]" if node_name else ""
        self.errors.append(f"Error{location}: {message}")
    
    def add_warning(self, message, node_name=None):
        location = f" [{node_name}]" if node_name else ""
        self.warnings.append(f"Warning{location}: {message}")
    
    @property
    def is_valid(self):
        return len(self.errors) == 0


class DialogueValidatorPlugin(PluginInterface):
    """Plugin that validates dialogues"""

    def __init__(self):
        super().__init__()
        self.plugin_info = PluginInfo(
            name="Dialogue Validator",
            version="1.0.0",
            description="Validates dialogues for common issues",
            author="Your Name",
            plugin_type=PluginType.DIALOGUE_PROCESSOR
        )
        self.menu_action = None
        self.current_dialogue = None

    def initialize(self, plugin_manager):
        return True

    def activate(self):
        return True

    def deactivate(self):
        if self.menu_action:
            self.menu_action.deleteLater()
        return True

    def get_hooks(self):
        return {
            PluginHooks.UI_MENU_BAR_CREATED: [self.on_menu_bar_created],
            PluginHooks.DIALOGUE_LOADED: [self.on_dialogue_loaded],
        }

    def on_menu_bar_created(self, menu_bar):
        for action in menu_bar.actions():
            if action.text() == "&Tools":
                tools_menu = action.menu()
                tools_menu.addSeparator()
                self.menu_action = tools_menu.addAction("Validate Dialogue")
                self.menu_action.triggered.connect(self.run_validation)
                break

    def on_dialogue_loaded(self, dialogue):
        self.current_dialogue = dialogue

    def run_validation(self):
        if not self.current_dialogue:
            from PyQt6.QtWidgets import QMessageBox
            QMessageBox.warning(None, "Validation", "No dialogue loaded")
            return

        result = ValidationResult()
        self._validate_dialogue_structure(result)
        self._validate_nodes(result)
        self._validate_options(result)
        self._validate_links(result)
        
        from PyQt6.QtWidgets import QMessageBox
        
        if result.is_valid:
            QMessageBox.information(None, "Validation Results", 
                "✓ Validation passed! No errors found.")
        else:
            msg = "✗ Validation failed:\n\n" + "\n".join(result.errors + result.warnings)
            QMessageBox.critical(None, "Validation Results", msg)

    def _validate_dialogue_structure(self, result):
        if not self.current_dialogue.npcname:
            result.add_error("NPC name is missing")
        
        if len(self.current_dialogue.nodes) == 0:
            result.add_error("Dialogue has no nodes")
            return

        has_starting = any(node.is_wtg for node in self.current_dialogue.nodes)
        if not has_starting:
            result.add_warning("No starting node (is_wtg) defined")

    def _validate_nodes(self, result):
        for i, node in enumerate(self.current_dialogue.nodes):
            if not node.nodename:
                result.add_error(f"Node {i} has no name")
            if not node.npctext.strip():
                result.add_warning("Node has no NPC text", node.nodename)

    def _validate_options(self, result):
        for node in self.current_dialogue.nodes:
            if len(node.options) == 0:
                result.add_warning("Node has no options", node.nodename)

    def _validate_links(self, result):
        node_names = {node.nodename for node in self.current_dialogue.nodes}
        for node in self.current_dialogue.nodes:
            for i, option in enumerate(node.options):
                if option.nodelink and option.nodelink not in node_names:
                    result.add_error(f"Option links to non-existent node", node.nodename)

6. API Reference Documentation

Plugin Interface

class PluginInterface
Base class that all plugins must inherit from.
__init__(self) -> None
Initialize the plugin. Subclasses must call super().__init__() and set self.plugin_info.
initialize(self, plugin_manager: PluginManager) -> bool
Called when plugin is first loaded. Use for one-time initialization.
activate(self) -> bool
Called when plugin becomes active. Use for enabling functionality.
deactivate(self) -> bool
Called when plugin is deactivated. Use for cleanup.
shutdown(self) -> None
Called when plugin is being unloaded. Final cleanup.
get_hooks(self) -> Dict[str, List[Callable]]
Return dictionary of hook handlers.

discover_plugins(self) -> List[PluginInfo]

7. Security Considerations

Input Validation

def process_user_input(self, user_input: str) -> str:
    # Validate input length
    if len(user_input) > 10000:
        raise ValueError("Input too long")
    
    # Sanitize for file operations
    sanitized = user_input.replace("..", "").replace("/", "_")
    
    # Validate characters
    if not sanitized.isalnum() and not "_" in sanitized:
        raise ValueError("Invalid characters")
    
    return sanitized

Path Traversal Prevention

import os

def safe_path_join(self, base_dir: Path, user_path: str) -> Path:
    target = (base_dir / user_path).resolve()
    try:
        target.relative_to(base_dir.resolve())
    except ValueError:
        raise SecurityError("Path traversal attempt detected")
    return target

Secure File Operations

def write_to_file(self, file_path: Path, content: str):
    # Validate file extension
    allowed_extensions = {'.json', '.txt', '.xml'}
    if file_path.suffix.lower() not in allowed_extensions:
        raise ValueError("Invalid file type")
    
    # Check file size
    if len(content.encode('utf-8')) > 10 * 1024 * 1024:
        raise ValueError("File too large")

"""

import xml.etree.ElementTree as ET

    def export(self):
        if not self.dialogue: return
        
        from PyQt6.QtWidgets import QFileDialog
        path, _ = QFileDialog.getSaveFileName(None, "Export XML", f"{self.dialogue.npcname}.xml")

        if path:
            xml = self.create_xml()
            open(path, 'w').write(xml)

    def create_xml(self) -> str:
        root = ET.Element("dialogue")
        root.set("exported", datetime.now().isoformat())
        
        meta = ET.SubElement(root, "metadata")
        ET.SubElement(meta, "npc").text = self.dialogue.npcname
        
        nodes = ET.SubElement(root, "nodes")
        for node in self.dialogue.nodes:
            node_el = ET.SubElement(nodes, "node")
            node_el.set("name", node.nodename)
            ET.SubElement(node_el, "text").text = node.npctext
            
            for opt in node.options:
                opt_el = ET.SubElement(node_el, "option")
                ET.SubElement(opt_el, "text").text = opt.optiontext

        ET.indent(root)
        return ET.tostring(root, encoding='unicode')

Example 2: Statistics Analyzer

"""
Statistics Plugin
Analyzes dialogue and provides statistics.
"""

from collections import Counter
from core.plugin_system import PluginInterface, PluginType, PluginHooks, PluginInfo


class StatisticsPlugin(PluginInterface):

    def __init__(self):
        super().__init__()
        self.plugin_info = PluginInfo(
            name="Dialogue Statistics",
            version="1.0.0",
            description="Analyze dialogue statistics",
            author="FDC Team",
            plugin_type=PluginType.DIALOGUE_PROCESSOR
        )
        self.dialogue = None

    def get_hooks(self):
        return {
            PluginHooks.UI_MENU_BAR_CREATED: [self.add_menu],
            PluginHooks.DIALOGUE_LOADED: [self.store_dialogue],
        }

    def add_menu(self, menu_bar):
        for action in menu_bar.actions():
            if action.text() == "&Tools":
                action.menu().addAction("Show Statistics").triggered.connect(self.show_stats)
                break

    def store_dialogue(self, dialogue):
        self.dialogue = dialogue

    def show_stats(self):
        if not self.dialogue: return

        total_nodes = len(self.dialogue.nodes)
        total_options = sum(len(n.options) for n in self.dialogue.nodes)
        
        skill_counts = Counter()
        for node in self.dialogue.nodes:
            for opt in node.options:
                if opt.has_skill_check and opt.skill_check:
                    skill_counts[opt.skill_check.check_what.name] += 1

        text = f"""Dialogue Statistics

NPC: {self.dialogue.npcname}
Total Nodes: {total_nodes}
Total Options: {total_options}

Skill Checks:"""
        
        for skill, count in skill_counts.items():
            text += f"\n  {skill}: {count}"

        from PyQt6.QtWidgets import QMessageBox
        QMessageBox.information(None, "Statistics", text)

14. Frequently Asked Questions

How do I create my first plugin? ▼

Create a new Python file in the plugins/ directory. Inherit from PluginInterface, define your PluginInfo, and implement the required methods. See Tutorial 1 for a complete example.

Where should I place my plugin files? ▼

Bundled plugins go in ./plugins/, user plugins go in ~/.fallout_dialogue_creator/plugins/.

How do I add a menu item to the application? ▼

Register for the UI_MENU_BAR_CREATED hook. In your handler, find the desired menu and add your action.

Can I access the current dialogue from my plugin? ▼

Yes! Register for the DIALOGUE_LOADED hook. The dialogue object will be passed to your handler.

How do I handle errors gracefully? ▼

Always wrap potentially failing operations in try/except blocks. Provide clear error messages to users via QMessageBox. Log errors for debugging.

Can plugins depend on other plugins? ▼

Yes, declare dependencies in your PluginInfo. Check for them in your initialize method.

How do I test my plugin? ▼

Use Python's unittest or pytest. Create mock objects for PluginManager, Dialogue, etc.

Why is my hook not being called? ▼

Check: 1) Hook is registered in get_hooks(), 2) Handler has correct signature, 3) Plugin is in ACTIVE state, 4) Hook name is correct.

How do I export data from the dialogue? ▼

Create an export plugin. Access the dialogue object, iterate through nodes/options, and write to your desired format.

What if my plugin needs to store settings? ▼

Use the application's Settings class. Call settings.get() to read and settings.set() to save.

Can I use external Python packages in my plugin? ▼

Yes, but consider portability. Document requirements clearly. Consider using the standard library when possible.
