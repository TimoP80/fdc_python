// PROJECT OVERVIEW

> Welcome to Fallout Dialogue Creator 2.0, a modern tool for creating and editing dialogue files for Fallout 1 and Fallout 2. Originally based on the classic FMF (Fan Made Fallout) format, this application has evolved into a full-featured dialogue editor with a distinctive Fallout 2 aesthetic.

The application is built using Python 3 and PyQt6, providing a cross-platform solution for modders and game developers who want to create authentic Fallout dialogue.

Note: This application is not affiliated with, nor endorsed by, Bethesda Softworks or Interplay Entertainment. Fallout is a trademark of Bethesda.

// KEY FEATURES

FMF File Encoding Detection

Automatically detects file encoding (UTF-8, UTF-16, Windows-1252, DOS code pages) for maximum compatibility with legacy dialogue files.

Fallout 2 Themed UI

Authentic post-apocalyptic aesthetic with olive drab greens, rust oranges, and signature yellow highlighting. CRT-style visual effects included.

Plugin Designer

Built-in visual plugin designer allows you to create custom plugins without writing code. Extend the application's functionality with your own additions.

Script Compiler Integration

Configure and use the SSL (Fallout Scripting Language) compiler to compile dialogue scripts directly from the application.

MSG/DDF Export

Export dialogues to multiple formats including SSL scripts, MSG message files, and DDF format for maximum compatibility.

Enhanced Diagram Widget

Visual dialogue editor with color-coded nodes, curved arrows, gradients, and smooth zooming for intuitive dialogue flow design.

Complete Menu System

Full menu bar with File, Edit, View, Dialogue, Tools, and Help menus. Comprehensive keyboard shortcuts for efficient workflow.

Plugin System

Extensible architecture with hot-reloadable plugins. Add new export formats, parsers, and UI extensions.

// INSTALLATION

Requirements

  • Python 3.9+ - Required runtime
  • PyQt6 - GUI framework
  • pyyaml - Configuration file handling

Dependencies Installation

pip install PyQt6 pyyaml

Running from Source

# Clone the repository
git clone https://github.com/TimoP80/fdc_python.git
cd fdc_python

# Install dependencies
pip install -r requirements.txt

# Run the application
python main.py

Running the Executable

If you have a pre-built executable (Windows), simply run: 

FalloutDialogueCreator.exe
Tip: On first run, the application will create a configuration directory at %APPDATA%/FMFDLG for storing settings and user data.

// USAGE GUIDE

Getting Started

  1. Launch the application - Either run python main.py or double-click the executable
  2. Create or open a dialogue - Use File → New or File → Open
  3. Add nodes - Right-click in the dialogue tree or use the toolbar
  4. Edit NPC text - Select a node and type in the NPC text field
  5. Add player options - Use the options section to add response choices
  6. Link nodes - Set the "Link to" field to connect options to other nodes
  7. Save your work - File → Save

Visual Diagram Editor

The diagram widget provides a visual representation of your dialogue flow:

  • Green nodes - Starting nodes (WTG - "What to say")
  • Blue nodes - Regular dialogue nodes
  • Gray nodes - Hidden nodes
  • Orange nodes - Nodes with skill checks
  • Gold highlight - Selected node
  • Arrows - Green (good), Red (bad), Gray (neutral) reactions

Exporting Dialogue

To export your dialogue:

  1. Go to File → Export
  2. Choose format: SSL, MSG, or DDF
  3. Select destination and filename
  4. Click Export

// CONFIGURATION

Settings Location

Configuration is stored in:

  • Windows: %APPDATA%/FMFDLG
  • Linux/Mac: ~/.config/FMFDLG

Configuration Options

SettingDescriptionDefault
font_sizeUI font size10
font_familyUI font familySegoe UI
themeColor theme (system/light/dark)system
auto_saveEnable auto-savefalse
script_compiler_pathPath to sslc.exe(default)
fo2_data_pathFallout 2 data directory(none)

Script Compiler Setup

To compile SSL scripts, you need the Fallout script compiler (sslc.exe):

  1. Obtain sslc.exe from a Fallout 2 modding source
  2. Go to Tools → Configure Script Compiler
  3. Browse to select sslc.exe
  4. Click Save
Warning: The plugin system allows arbitrary Python code execution. Only install plugins from trusted sources.
